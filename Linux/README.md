# Linux Commands

### Check if program is installed

```
which <program> 
```

### Manual page for program 

```
man <program>
```

### Print Y forever

```
yes
```

### Clear Terminal 

```
ctrl + l 
```

### Linux Task Manager

```
top 
```

OR 

```
htop 
````

### Kill Process

```
exit
```

OR 

ctrl D
